1. Objective

The objective of this assignment is to build and deploy a static personal blog website. The website is used to present course-related work and demonstrate the use of Git for version control and GitHub Pages for deployment.

2. Tools and Technologies

Tool Purpose
HTML Structure of the website
CSS Styling of the website
Git Version control
GitHub Remote repository hosting
GitHub Pages Website deployment
Markdown Documentation

3. Website Development Process

The development process was carried out step by step.

First, a project folder was created for the website. Then, an index.html file was written to define the structure of the webpage. A CSS file was added to improve the appearance and layout.

After that, the Markdown and PDF files from Assignment 1 were added to the project and linked on the homepage. This ensures integration of previous work.

Finally, the Assignment 2 documentation was created and linked to the homepage.

4. Git Version Control Process

Git was used to manage the project with meaningful commits.

Each commit reflects a specific step in the development process:

  • Initial website homepage: Created the basic HTML structure
  • Add basic CSS styling: Improved visual appearance
  • Add Assignment 1 report files: Integrated previous work
  • Add Assignment 2 documentation: Added documentation file
  • Update homepage with assignment links: Linked all resources
  • Add deployment note to homepage: Explained deployment method

This approach ensures a clear and logical development history.

5. Deployment

The website was deployed using GitHub Pages.

GitHub Pages was chosen because it is simple to use and directly supports static website hosting from a GitHub repository.

After pushing the project to GitHub, the Pages feature was enabled in the repository settings, and the site became accessible via a public URL.

6. Integration of Previous Work

The homepage includes links to Assignment 1 Markdown and PDF reports.

These files can be accessed directly from the website, satisfying the requirement of integrating previous work.

7. Conclusion

Through this assignment, I learned how to build a static website using HTML and CSS, manage the development process using Git, and deploy the website using GitHub Pages.

This project also improved my understanding of version control and technical documentation.
